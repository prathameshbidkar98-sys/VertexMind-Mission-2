# Mission 2 - Database Integration & CRUD API

## Features
- Create User
- Get All Users
- Get User By ID
- Update User
- Delete User

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## API Endpoints

POST   /api/users
GET    /api/users
GET    /api/users/:id
PUT    /api/users/:id
DELETE /api/users/:id

## Installation

npm install

## Run Project

npm start
